# FlexAI - Мобильное приложение с прокси

🤖 **FlexAI** - это веб-приложение-прокси для доступа к сервису FlexAI из России. Приложение автоматически обходит блокировки и скрывает водяные знаки Lovable.

## ✨ Особенности

- 🔓 **Автоматический прокси** - Использует несколько CORS-прокси для надежного доступа
- 🚫 **Без водяных знаков** - Автоматически удаляет все упоминания Lovable
- 📱 **Мобильная оптимизация** - Адаптированный интерфейс для мобильных устройств
- 🔄 **Авто-переключение** - При проблеме с одним прокси автоматически переключается на другой
- ⚡ **Быстрая загрузка** - Оптимизированный код без серверной части
- 🎨 **Современный UI** - Красивый градиентный дизайн в стиле мобильного приложения

## 🚀 Быстрый старт

### Вариант 1: Локальный запуск

Просто откройте файл `index.html` в любом современном браузере:

```bash

```

```bash

npx http-server -p 8000
```

Затем откройте в браузере: `http://localhost:8000`

## 🔧 Как это работает

1. Приложение загружает целевой сайт `flexai-ru.lovable.app` через публичные CORS-прокси
2. После загрузки автоматически удаляются все элементы с классами/ID содержащими "lovable" или "watermark"
3. Инжектится CSS для скрытия любых водяных знаков
4. При ошибке загрузки автоматически переключается на следующий прокси из списка

### Используемые прокси:

- `api.allorigins.win`
- `corsproxy.io`
- `thingproxy.freeboard.io`

## ⚠️ Важные замечания

1. **CORS ограничения**: Некоторые функции могут быть ограничены политиками безопасности браузера
2. **Производительность**: Использование прокси может незначительно увеличить время загрузки
3. **Доступность**: Если все прокси недоступны, появится экран ошибки с кнопкой повторной попытки

## 🔒 Безопасность

- Приложение не хранит никакие данные пользователей
- Не используется серверная часть - весь код выполняется в браузере
- HTTPS соединение с целевым сайтом
- Песочница (sandbox) для iframe для дополнительной безопасности
